## What is Shell?
The shell is program that acts as an interface/mediator between you(user) and the operating system, allowing users to enter commands and execute programs. Commands entered into the shell are executed immediately, and the results are returned in the same window.


CLI - Command Line Interface
GUI - Graphical User Interface

Upon logging, you are placed in your home directory, typically located under `/home`. For example:
As my username is `rimonmostafiz`, my home directory is at `/home/rimonmostafiz`.
User Alen and Bob will have a home directory at `/home/alen` and `/home/bob`.
